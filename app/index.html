Escape is an inspirational air shopping application. It uses several of REST APIs from Sabre Dev Studio.

Search - Calls the Destination Finder API

The search parameters are used to consume a Destination Finder API request, and display results on a Google Maps.
https://developer.sabre.com/docs/read/rest_apis/air/search/destination_finder

By clicking later arrow under the graph the app makes a call to Lead Price Callendar to get fares info for the next 192 days.
https://developer.sabre.com/docs/read/rest_apis/air/search/lead_price_calendar

The seasonality indicator is based on the destination and date selected and Seasonality API
https://developer.sabre.com/docs/read/rest_apis/air/intelligence/travel_seasonality.

If US Point of Sale Country has been selected app provides Advice Indicator based on Low Fare Forecast API Call for Origin/Destination and specific Date selected.
https://developer.sabre.com/docs/read/rest_apis/air/intelligence/low_fare_forecast

The view flights button makes a call to to InstaFligh Search API and provides flight info for top ten connections https://developer.sabre.com/docs/read/rest_apis/air/search/instaflights_search
